Cargo space optimization
This is the final process of optimization. At this stage, the list of packages has been optimized and indicates which ones are more beneficial to load onto the vehicle. To complete the process, the user must specify the vehicle's properties so that the algorithm can perform cargo space optimization calculations.
Cargo space dimensions
Enter the dimensions of the available cargo space. Please note that this tool is currently not capable of performing calculations that account for curved or skewed vehicle walls. Therefore, the dimensions must represent a cuboid-shaped space. Length [cm] Width [cm] Height [cm]
Load weight
Enter the maximum allowed weight. Note that this value refers only to the weight of the load and does not include the vehicle's own weight. The user is responsible for adjusting the load weight accordingly to comply with legal requirements and account for the vehicle's weight. Maximum allowed weight [t]
Optimize load
Once all required data has been entered, the final optimization process can begin. Press the button below to start the calculations and run the package placement simulation.
